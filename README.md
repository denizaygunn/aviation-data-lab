## 1. Predictive Maintenance

**Goal:**  
To build a simple predictive maintenance model that can detect potential failures in aircraft components using sensor data.

**Description:**  
This module demonstrates a machine learning pipeline using simulated temperature, pressure, and vibration data to classify whether a failure event occurs.

**Technologies:**  
- Python  
- Pandas  
- scikit-learn  
- Random Forest Classifier  

**How to Run:**  
```bash
cd maintenance_prediction/src
python3 model_train.py
